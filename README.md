# Homebrew Tap for Cloudflare Status Bar

A macOS menu bar app for monitoring your Cloudflare resources (Workers, Pages, KV, R2, D1, Queues).

## Installation

```bash
brew tap sushaantu/cloudflare-status-bar
brew install --cask cloudflare-status-bar
```

## Requirements

- macOS 13.0 (Ventura) or later
- [Wrangler CLI](https://developers.cloudflare.com/workers/wrangler/) authenticated (`wrangler login`)
